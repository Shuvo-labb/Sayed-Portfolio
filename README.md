# MD Abu Sayed Shuvo - Portfolio

Modern, interactive portfolio website built with React, GSAP, and Vite.

## Features

- Modern dark theme with purple accent gradients
- GSAP-powered smooth animations and scroll triggers
- Custom animated cursor
- Fully responsive design
- Hero section with floating particles
- Skills section with modern pill tags
- Project showcase cards
- Education timeline
- Contact form with Resend email API
- Downloadable CV (PDF)
- LinkedIn, GitHub social links

## Tech Stack

- React 18
- Vite
- GSAP (GreenSock Animation Platform)
- React Icons
- Resend (Email API)
- Vercel (Hosting + Serverless Functions)
- CSS3
- Github Copilot

## Getting Started

```bash
npm install
npm run dev
```
