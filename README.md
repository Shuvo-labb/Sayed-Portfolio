# MD Abu Sayed Shuvo - Portfolio

Modern, interactive portfolio website built with React, GSAP, and Vite.

## Features

- Modern dark theme with purple accent gradients
- GSAP-powered smooth animations and scroll triggers
- Custom animated cursor
- Fully responsive design
- Hero section with floating particles
- Skills section with progress bars
- Project showcase cards
- Education timeline
- Contact form
- Integrated CV viewer

## Tech Stack

- React 18
- Vite
- GSAP (GreenSock Animation Platform)
- React Icons
- CSS3

## Getting Started

```bash
npm install
npm run dev
```
